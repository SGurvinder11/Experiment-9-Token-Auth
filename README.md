# Experiment 9 – Token Based Authentication using Flask

## Description
This experiment demonstrates token based authentication using Flask APIs.

## Authentication Methods Implemented
1. Basic Authentication using Authorization Header
2. Custom Header Authentication
3. JWT (JSON Web Token) Authentication

## Technologies Used
- Python
- Flask
- PyJWT
- Postman
- Render

## API Endpoints

| Endpoint | Method | Description |
|--------|--------|-------------|
| /auth/basic | GET | Basic Authentication |
| /auth/custom | GET | Custom Header Authentication |
| /login | POST | Generate JWT Token |
| /auth/jwt | GET | Access using JWT Token |

## Learning Outcomes
- Understand token based authentication
- Implement Basic Authentication
- Implement Custom Header Authentication
- Implement JWT authentication
- Test APIs using Postman
- Deploy backend application on Render
